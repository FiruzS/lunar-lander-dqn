# 📊 Результаты работы Lunar Lander DQN

## Цель проекта
Реализация Deep Q-Network (DQN) агента для игры Lunar Lander из gymnasium с тремя режимами:
1. Игра человеком
2. Обучение нейросети
3. Игра обученного компьютера

---

## 🎯 Реализованный функционал

### ✅ Три режима работы

**1. Human Mode** - интерактивная игра
```bash
python main.py --mode human
```

**2. Training Mode** - обучение DQN агента
```bash
python main.py --mode train --episodes 500 --no-render
```

**3. Computer Mode** - демонстрация обученной модели
```bash
python main.py --mode computer --model model.pth --games 5
```

---

## 🧠 Архитектура решения

### Нейронная сеть
```
Input (8) → Linear(64) + ReLU → Linear(64) + ReLU → Output (4)
```

**Параметры:**
- Входной слой: 8 нейронов (состояние среды)
- Скрытые слои: 2 × 64 нейрона с ReLU активацией
- Выходной слой: 4 нейрона (Q-значения для действий)

### Гиперпараметры обучения

| Параметр | Значение |
|----------|----------|
| Learning Rate | 0.0005 |
| Gamma (discount) | 0.99 |
| Epsilon Start | 1.0 |
| Epsilon End | 0.01 |
| Epsilon Decay | 0.995 |
| Batch Size | 64 |
| Memory Size | 10000 |
| Target Update | 10 эпизодов |

- **PyTorch** - фреймворк для глубокого обучения

1. **Experience Replay** - сохранение опыта в буфер

3. **Target Network** - стабильные целевые значения

## 🔍 Анализ архитектуры

### Почему выбрана такая архитектура?

**2 скрытых слоя по 64 нейрона:**
- Достаточно для аппроксимации Q-функции
- Быстрое обучение (~40 минут)
- Хорошая сходимость без переобучения

**ReLU активация:**
- Решает проблему затухающих градиентов
- Стандарт для таких задач
- Быстрое вычисление

**Adam оптимизатор:**
- Адаптивный learning rate
- Устойчивость к шумным градиентам
- Хорошая сходимость из коробки

## 🎮 Критерии успеха в Lunar Lander

### Начисление очков

- ✅ Вертикальное положение (+30-50)
